# Paper Directory

This directory contains the active arXiv preprint manuscript.

- `main.pdf`: latest compiled paper.
- `main.tex`: manuscript source.
- `main.bbl`: generated bibliography file kept for arXiv source upload.
- `references.bib`: bibliography.

Temporary LaTeX files such as `.aux`, `.blg`, `.log`, and `.out` are not kept
here after cleanup. Rebuilding the PDF will regenerate them.
